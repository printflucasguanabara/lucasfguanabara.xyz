Este projeto nasceu da paixão pelos jogos digitais e da vontade de criar uma comunidade de jogadores entusiastas.

O projeto representa um marco pessoal, pois é onde apliquei extensivamente meus conhecimentos em **CSS**. 

Por fim, o resultado é um design/template que captura a essência do que imagino para um blog sobre jogos - um espaço onde design intuitivo e conteúdo interativo se encontram para criar uma experiência imersiva para os amantes de jogos.
